# 🎯 Quiz Game

A simple and interactive Quiz Game built using HTML, CSS, and JavaScript.

The game randomly selects 5 questions from the question bank, allows users to answer them one by one, and displays the final score at the end.

---

## 🚀 Features

- Randomly selects 5 questions
- Multiple choice answers
- Instant answer validation
- Score tracking
- Next Question functionality
- Final score display
- Responsive UI
- Beginner-friendly JavaScript project

---

## 🛠️ Technologies Used

## 🎮 How It Works

1. The quiz starts automatically.
2. Five random questions are selected.
3. Select an answer.
4. Correct answers increase your score.
5. Click "Next Question" to continue.
6. Final score is displayed after all questions are completed.
